## Getting Started

### clone

```bash
$ git clone git@github.com:yuikoito/saboritai.git
```

### install

```bash
$ yarn install
```

### start

```bash
$ yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
